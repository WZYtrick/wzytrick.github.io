数组 链表 实现线性表

1.Array

​ A kind of data structure which occupies a continuous memory space. The elements of it are accessed by the array subscripts(index). When using the array to implement Linear List , we should pay attention to the followings.

  • Because the length of array is fixed , adding elements should judge the size. When the size is not enough , double the array size.
  • When deleting and inserting , the elements followed the modified index should be moved.
