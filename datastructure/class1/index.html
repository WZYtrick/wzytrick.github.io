数组 链表 实现线性表

1.Array

​ A kind of data structure which occupies a continuous memory space. The elements of it are accessed by the array subscripts(index). When using the array to implement Linear List , we should pay attention to the followings.

  • Because the length of array is fixed , adding elements should judge the size. When the size is not enough , double the array size.
  • When deleting and inserting , the elements followed the modified index should be moved.

2.Linked List

​ linked list is a linear collection of data elements whose order is not given by their physical placement in memory. Instead, each element points to the next. It is a data structure consisting of a collection of nodes which together represent a sequence. In its most basic form, each node contains: data, and a reference (in other words, a link) to the next node in the sequence.

一、逆序构造单链表

​ 设一个指针pre初始指向NULL，每次建立一个节点把这个节点的next指针指向pre，然后把pre指向这个创建的节点。完成后把头节点next指针指向pre。

二、单链表reverse(转置/颠倒)

​ 头结点的next和next的next为空直接return，不为空顺序遍历该链表通过逆序构造单链表的方式完成转置。
