Relation Algebra

Basic operators

  • select 选择 σ

    根据给出的条件选出一系列元组

    e.g. σ~A=B^D>5~(r) 表示从关系r中选出属性A等于属性B的值且属性D的值大于5的元组set(这里set表示无重复元素的集合)。

  • project 投影 π

    选取指定属性的列的集合

    e.g. π~A,C~(r) 表示从关系r中提取出A，C两属性的列。
