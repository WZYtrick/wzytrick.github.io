Relation Algebra

Basic operators

  • select 选择 σ

    根据给出的条件选出一系列元组，其下标表示断言(predicate)

    e.g. σA=B^D>5(r) 表示从关系r中选出属性A等于属性B的值且属性D的值大于5的元组set(这里set表示无重复元素的集合)。

  • project 投影 π

    选取指定属性的列的集合

    e.g. πA,C(r) 表示从关系r中提取出A，C两属性的列。

  • union 取并集 υ

    如同集合一样对两个relation取并集

    e.g. A υ B表示两个关系取并，无重复元素，这里要求A与B两个关系具有相同的属性且他们的域(domain)也相同。

  • set difference 差集 -

    A-B=!B∩A

  • cartesian product 笛卡尔积 ×

    r :

    A B
    a 1
    b 2

    s :

    A C
    a 5
    c 8

    r × s ：

    A B A C
    a 1 a 5
    a 1 c 8
    b 2 a 5
    b 2 c 8

  • rename 重命名 ρ

    写为：ρname(A1,A2…)(E)

    其中name是重命名后的名字，A123…是属性（也可以不写），E是要被重命名的表达式。

  • set intersection 交集 ∩

    r ∩ s = r - (r - s)

  • natural join 自然连接 ⋈

    r ：

    A B C D
    …

    s :

    C D E F
    …

    r ⋈ s :

    A B C D E F
    …

    其中只有共同属性相同的元组会相连。

  • assignment 赋值 ←

    e.g. temp1←E

  • division 除 ÷

    定义：r ÷ s = {t | t∈πr-s(r) ∧ ∀u∈s(t×u∈r)}

    e.g.

    r :

    A B
    a 1
    a 2
    b 3
    b 4
    c 2

    s ：

    B
    1
    2

    r ÷ s ：

    A
    a

  • generalized projection 广义投影

    可在属性位置写一些运算式

    e.g. πA,B-C(r) 这里-代表B，C两属性相减运算
